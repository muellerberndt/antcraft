# AntCraft Hotkey Reference

## Mouse

| Action               | Control                |
|----------------------|------------------------|
| Select unit          | Left click             |
| Select multiple      | Left click + drag box  |
| Context command      | Right click (auto-detects target: enemy=attack, corpse=harvest, hive site=found, ground=move) |

## Keyboard

| Key            | Action                                              |
|----------------|-----------------------------------------------------|
| S              | Stop selected units                                 |
| M              | Move mode — next right-click issues Move command    |
| A              | Attack mode — next right-click issues Attack command|
| E              | Harvest mode — next right-click harvests a corpse   |
| F              | Found mode — next right-click founds hive at site   |
| N              | Spawn ant from selected hive (or nearest own hive)  |
| Q              | Merge selected ants into a queen at nearest hive    |
| H              | Send selected queen to found hive at nearest site   |
| Up Arrow       | Scroll camera up                                    |
| Left Arrow     | Scroll camera left                                  |
| Down Arrow     | Scroll camera down                                  |
| Right Arrow    | Scroll camera right                                 |
| F11            | Toggle fullscreen                                   |
| ESC            | Quit game                                           |

## Debug Info

The top-left overlay shows:
- **Tick** -- Current simulation tick
- **Player** -- Your player ID (0 or 1)
- **Jelly** -- Your current jelly stockpile
- **Ants** -- Your ant count
- **FPS** -- Frames per second
- **Connected** -- Network connection status
- **Selected** -- Number of selected units
