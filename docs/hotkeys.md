# AntCraft Hotkey Reference

## Mouse

| Action               | Control                |
|----------------------|------------------------|
| Select unit          | Left click             |
| Select multiple      | Left click + drag box  |
| Move selected units  | Right click            |

## Keyboard

| Key            | Action                    |
|----------------|---------------------------|
| S              | Stop selected units       |
| W / Up Arrow   | Scroll camera up          |
| A / Left Arrow | Scroll camera left        |
| S / Down Arrow | Scroll camera down        |
| D / Right Arrow| Scroll camera right       |
| F11            | Toggle fullscreen         |
| ESC            | Quit game                 |

## Planned (Not Yet Bound)

These commands exist in the simulation but don't have hotkeys yet:

| Command        | Description                                      |
|----------------|--------------------------------------------------|
| Spawn Ant      | Spawn an ant from a hive (costs 10 jelly)        |
| Merge Queen    | Merge 5 ants at a hive into a queen              |
| Found Hive     | Send a queen to claim a hive site                |
| Harvest        | Send ants to harvest jelly from a corpse         |

## Debug Info

The top-left overlay shows:
- **Tick** -- Current simulation tick
- **Player** -- Your player ID (0 or 1)
- **Jelly** -- Your current jelly stockpile
- **Ants** -- Your ant count
- **FPS** -- Frames per second
- **Connected** -- Network connection status
- **Selected** -- Number of selected units
